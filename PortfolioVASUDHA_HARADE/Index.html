Online Event Management System

Description: This is a full-stack web application that helps users manage events efficiently — from event registration to planning and post-event reporting. The goal is to simplify event logistics for organizers and attendees.

Features:

  • User Registration and Login System
  • Create, Update, and Delete Events (CRUD)
  • Event Details Page with Date, Time, Venue, and Description
  • Online Event Registration for Participants
  • Admin Dashboard to Manage Users and Events
  • Event Reporting — View Participants, Download Reports

Technology Used:

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Java (JSP, Servlets)
  • Database: MySQL
  • Tools: NetBeans IDE, MySQL Workbench

Impact: This system saves time for event organizers by automating the event process, improves participant experience with online registration, and provides insightful reports. Ideal for colleges, companies, or communities hosting regular events.
